## 1. 작업 요약

레거시 PICenter_RM (JSP 기반)의 화면 디자인을 PICenter_TOTALS의 HTML 페이지에 그대로 반영하는 작업을 수행했습니다.

### 변경 범위
- HTML 페이지 10개 전체 재개발 (레거시 디자인 적용)
- 네비게이션 메뉴 일관성 통일
- 로고 텍스트 통일: "개인정보 검출관리 센터"
- User 엔티티 및 인증 서비스 DB 스키마 정합성 수정
- 서버 라우팅 설정 보완

## 2. HTML 페이지 변경 내역

### 2.1 login.html (로그인 페이지)
- 레거시 2-column 레이아웃 적용 (좌: 다크 #262c38, 우: 흰색)
- "개인정보 검출관리 센터" 로고 표시
- REST API 로그인: `POST /api/v1/auth/login`
- 쿠키 기반 아이디 저장 기능
- 비밀번호 변경 팝업

### 2.2 picenter_dashboard.html (대시보드)
- 레거시 3-column 레이아웃 (트리 351px + 도넛차트 413px + 바차트)
- ECharts 5 CDN 도넛/바 차트
- 랭킹 테이블 (AG Grid 스타일)
- API 연동: `/api/v1/dashboard/system-current`, `/rankings`, `/charts/bar`

### 2.3 picenter_detection.html (검출 결과 조회)
- 좌측 트리 사이드바 (320px) + 우측 데이터 그리드
- 검색 폼 (호스트명, 기간별, 데이터유형)
- 상세 팝업 (검출 상세, 하위경로)
- API 연동: `/api/v1/detections`, `/{findId}`, `/subpaths/{targetId}`

### 2.4 picenter_target.html (타겟 관리)
- 검색 테이블 + 데이터 그리드
- 그룹/서버 목록 조회
- API 연동: `/api/v1/targets`, `/groups`, `/servers`

### 2.5 picenter_scan.html (스캔 관리)
- 탭 기반 레이아웃 (스케줄/스캔이력/스캔정책)
- 검색 테이블 + 데이터 테이블 + 페이지네이션
- API 연동: `/api/v1/scans/schedules`, `/history`, `/policies`

### 2.6 picenter_report.html (통합 보고서)
- 검색 테이블 (호스트명, 기간별, 경로)
- 동적 패턴 컬럼 데이터 그리드
- 다운로드 버튼
- API 연동: `/api/v1/reports/summary`

### 2.7 picenter_statistics.html (통계)
- 검색기간 필터 + 2-column 레이아웃
- 좌: ECharts 검출 현황 차트
- 우: TOP5 테이블 3개 (전체/정탐/오탐)
- API 연동: `/api/v1/statistics`, `/detections`, `/true-positives`, `/false-positives`

### 2.8 picenter_user.html (사용자 관리)
- 로고 텍스트 변경: "PICenter TOTALS" → "개인정보 검출관리 센터"

## 4. 백엔드 변경 내역

### 4.1 WebMvcConfig.java
- `/` → `login.html` 포워딩 추가
- `/login` → `login.html` 포워딩 추가

### 4.2 SecurityConfig.java
- `/login` 경로 permitAll 추가

### 4.3 User.java (엔티티)
- `picenter` DB의 `pi_user` 테이블 스키마에 맞게 복원
- `user_no`: PK (숫자)
- `user_id`: 로그인 ID (예: admin)
- `lock_status`: 계정 잠금 상태 (INT)
- `acc_yn`: 계정 활성화 여부 (VARCHAR)

### 4.4 UserServiceImpl.java
- `findByUserId(username)`: 로그인 ID로 사용자 조회
- 비밀번호 검증: `SHA2(HEX(password), 256)` 방식
- 실패 횟수 관리 및 계정 잠금 (5회 초과)
