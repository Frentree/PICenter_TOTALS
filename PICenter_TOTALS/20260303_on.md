## 2. TargetMapper.xml selectTargetUsers 쿼리 수정

**작업일**: 2026-03-03

### 2-1. 문제점
- `GET /api/v1/targets/{targetId}/users` API 호출 시 500 에러
- 원인: TargetMapper.xml의 selectTargetUsers 쿼리에서 pi_user 테이블의 잘못된 컬럼명 사용
  - `u.TEAM_CODE` → 실제 컬럼: `u.insa_code`
  - `u.EMAIL` → 실제 컬럼: `u.user_email`
  - `u.PHONE` → 실제 컬럼: `u.user_phone`
  - `u.USE_YN` → 실제 컬럼: `u.acc_yn`
  - `u.LAST_LOGIN_DT` → 실제 컬럼: `u.logindate`
  - `u.REG_DT` → 실제 컬럼: `u.regdate`

### 2-2. 수정 내용
- `selectTargetUsers` 쿼리의 컬럼명을 실제 DB 컬럼명에 맞게 수정 (AS alias 유지)
